# household expense tracker experiment

## Project Description
 An experimental frontend application to track and calculate monthly utility bills and household expenses. This project serves as a practical sandbox to learn web layout structure, responsive design, and data presentation.

## Core Features
* **Responsive Layout:** Designed to scale between desktop monitors and mobile displays.
* **Local Workspace Flow:** Built in VS Code and deployed using Git.

## Technical Toolset
* **Structure:** HTML5
* **Styling:** CSS3 
* **Logic:** JavaScript (Basics)
* **Deployment & Hosting:** Vercel
